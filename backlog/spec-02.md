# Spec 02

Navam Sentinel should be able to work with following model sources in priority order. Determine a plug and play, standardized way to add more model soruces.
1. Anthropic API
2. OpenAI API
4. Amazon Bedrock API
5. HuggingFace API
6. Ollama API

Navam should be able to handle multiple agentic frameworks in priority order. Determine a plug and play, standardized way to add more agentic frameworks.
1. LangGraph - https://www.langchain.com/langgraph
2. Claude Agent SDK - https://docs.claude.com/en/docs/agent-sdk/overview
3. OpenAI Agents SDK - https://openai.github.io/openai-agents-python/
4. Strands Agents - https://strandsagents.com/latest/
